Why Crawler?¶

  • 原始網站做得太爛，乾脆自己爬出資料做一個（萌典、ZIPCodeTW）
  • 資料分析（whoscall）
  • 搜尋引擎（Google, Yahoo）
  • 整合型網站（表特N大、比價網、翻書客、feedly）
  • 無聊，好玩，練功

爬蟲簡易概念¶

  1. 分析網站結構
    • static
    • dynamic (ajax)
  2. 取得需要的資料 - use python
    • 取得網頁原始碼 - requests
    • 取得需要的資訊 - pyquery
  3. 儲存結果
    • csv
    • json
    • mongodb (pymongo)
  4. 定時更新資料 (optional)

小技巧¶

  • 觀測表格(form)的傳送資訊
  • Developer Tools - Network

def get_article_data(article_url):                                           
    url = 'http://www.ptt.cc/bbs/Beauty/' + article_url + '.html'            
                                                                             
    s = get_requests_data(url)                                               
                                                                             
    data = {}                                                                
    metadata = s('.article-meta-value')                                      
    if len(metadata) != 4:                                                   
        return data                                                          
                                                                             
    urls = s('#main-content > a').map(lambda i, e: pq(this).text())     
    pic_urls = []                                                            
    for url in urls:                                                         
        if url.endswith('.gif'):
            continue                                                         
                                                                             
        if url.endswith('.jpg') or url.endswith('.png'):                     
            pic_urls.append(url)                              
                                                                             
    date = metadata.eq(3).text()                                                      
                                                                             
    if pic_urls:                                                             
        data['pic'] = pic_urls                                               
        data['date'] = date                                                  
    return data
